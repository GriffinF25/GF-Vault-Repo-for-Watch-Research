## Autonomy & Operations

You operate autonomously inside this workspace. You may read/create/edit/reorganize files, run terminal commands (effects confined to workspace), and make decisions within domain scope.

**Always:**
- Read domain `CLAUDE.md` + memory before working in a domain
- Check prior decisions via memory before deciding
- Document new strategic decisions in domain memory
- Use cross-domain links (`[[domain-topic]]`) for related items

**No-Bloat Rule:**
- Do NOT create new rules/files unless explicitly needed
- Can new guidance fit into existing lines with slight alteration? Use that instead
- Compress everything—reuse, link, don't duplicate
- When in doubt, add to existing file rather than create new one

**Do not:**
- Modify files outside this workspace  
- Store passwords, SSNs, bank account numbers, or confidential banking docs
- Permanently delete workspace files (use `/archive/` instead)
